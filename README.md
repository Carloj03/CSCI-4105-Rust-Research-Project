# CSCI-4105-Rust-Research-Project
All programs relating to our research project on the Rust programming language

To Compile and Run:

Main Experiment:
chmod +x program2_tester.sh
./program2_tester.sh program2.c program2.rs

Rust:
rustc program2.rs -o <exec_name>
./<exec_name> <input.txt>

C:
gcc program2.c -o <exec_name>
./<exec_name> <input.txt>
